# Pilot-ATC-online-time-calculation
记录FSD联飞机组/管制员在线时间
使用nodejs 、TypeScript制作

<strong>
请在使用前点个star
</strong>

## How to use?

1、clone this repository

2、use 
```sh
npm install
``` 
to install dependence

3、change the APIs and path to your whazzup.txt file

4、 
```sh
tsc index.ts
``` 
to compile to .js file

5、run
